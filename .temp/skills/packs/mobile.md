# Mobile Skill Pack

Use this pack for mobile agents working on Expo React Native setup,
implementation, and verification.

## Skills

- `okan-react-native-app-initializer`: initialize or normalize an Expo React
  Native app.
- `okan-figma-to-mobile-implementation-agent`: implement mobile UI from Figma
  designs.
- `okan-mobile-design-system-sync-agent`: sync Figma design-system foundations
  into mobile code.
- `okan-nativewind-best-practices-agent`: write, review, or fix NativeWind
  usage.
- `okan-mobile-accessibility-audit-agent`: audit or remediate mobile
  accessibility.
- `okan-mobile-qa-smoke-test-agent`: plan or run focused mobile QA smoke tests.
- `okan-expo-release-checklist-agent`: audit Expo/EAS release readiness.
- `okan-eas-build-submit-operator`: operate EAS build and submit workflows when
  explicitly requested.
- `okan-push-notification-implementation-agent`: implement or audit push
  notification support.
- `okan-app-onboarding-flow-designer`: plan or design onboarding flows.
- `caveman`: optional terse communication mode only when the user asks for it.

## Notes

- Use the assigned folder's local `AGENTS.md` before invoking a skill.
- Keep mobile implementation inside the assigned mobile folder unless the PM
  expands the boundary.
- Do not use this pack as automatic routing logic.
