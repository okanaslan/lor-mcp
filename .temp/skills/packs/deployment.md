# Deployment Skill Pack

Use this pack for deployment planning and provider configuration review.

## Skills

- `okan-render-deployment-checker`: inspect or validate existing Render
  services, deploys, databases, environment variables, logs, health checks, and
  deployment drift.
- `find-skills`: discover additional installable deployment skills only when
  needed.

## References

- `.temp/deploy-agents.md`: deployment agent rules.

## Notes

- Do not add provider tools, MCP servers, Terraform, Cloud Build, or deploy
  automation without explicit approval.
- Do not use this pack as automatic routing logic.
