# Design Skill Pack

Use this pack for design agents working on Figma-first app design, design
systems, handoff, and public-facing visual materials.

## Skills

- `okan-figma-product-screen-designer`: design or update in-app Figma screens,
  states, and handoff notes.
- `okan-figma-design-system-curator`: work on Figma foundations such as colors,
  typography, spacing, radius, elevation, and variables.
- `okan-figma-component-library-maintainer`: maintain reusable Figma component
  libraries, variants, and component hygiene.
- `okan-figma-auto-layout-specialist`: audit or repair Auto Layout structure
  and responsive sizing behavior.
- `okan-figma-layer-naming-organizer`: clean up layer names and selected-node
  hierarchy for handoff readability.
- `okan-landing-page-design-agent`: design public app landing pages in Figma.
- `okan-marketing-screenshot-designer`: design App Store and Google Play
  screenshot sets.
- `okan-store-listing-copy-agent`: write or audit app store listing copy.

## Notes

- Figma plugin skills are tool-driven workflow skills, not personal skills to
  list as installable pack entries.
- Use `.temp/design-agents.md` for design-system guidance when a separate design
  workspace exists.
- Do not use this pack as automatic routing logic.
