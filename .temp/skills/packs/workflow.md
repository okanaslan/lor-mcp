# Workflow And Review Skill Pack

Use this pack for implementation process, reviews, commits, pull requests, and
short operational communication.

## Skills

- `okan-codex-workflow`: follow the general Codex execution workflow.
- `okan-task-write-process`: implement planned changes or write source-of-truth artifacts.
- `okan-task-review-process`: review branches, pull requests, or working changes.
- `okan-code-review`: produce strict, actionable code review findings.
- `okan-engineering-review-checklist`: run a broad readiness and completeness pass.
- `okan-create-review-comments`: turn selected findings into inline review comments.
- `okan-commit-message`: create or suggest scoped conventional commits.
- `okan-pr-description`: generate concise pull request descriptions.
- `okan-respond-process`: draft short operational and review responses.

## Notes

- Use `.temp/review-agents.md` for review behavior and finding format.
- Use `.temp/process-agents.md` for concise engineering communication.
- Select only the skill matching the requested workflow.
- Do not treat this pack as automatic routing or permission to commit, push, or
  post review comments.
