# Frontend Skill Pack

Use this pack for frontend agents working on React, Vite, Tailwind, web UI, and
frontend verification.

## Skills

- `okan-frontend-marketing-website-agent`: implement public-facing React
  marketing and informational websites.
- `okan-figma-to-react-implementation-agent`: translate Figma designs into
  browser React code.
- `okan-tailwindcss-usage-agent`: write, review, or fix Tailwind CSS usage.
- `okan-react-accessibility-review-agent`: audit or remediate browser React
  accessibility.
- `okan-seo-landing-page-agent`: plan, write, implement, or audit SEO landing
  pages.
- `vercel-react-best-practices`: review React or Next.js performance and code
  quality.
- `find-skills`: discover additional installable frontend skills only when
  needed.
- `caveman`: optional terse communication mode only when the user asks for it.

## Notes

- Use the assigned folder's local `AGENTS.md` before invoking a skill.
- For Figma-to-frontend work, use the approved Figma tooling separately.
- Do not use this pack as automatic routing logic.
