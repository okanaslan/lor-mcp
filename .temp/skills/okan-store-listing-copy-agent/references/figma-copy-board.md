# Figma Copy Board Guidance

Use this when store listing copy should be recorded in Figma.

## Board Structure

- Create or update a review frame named by purpose, such as `Store Listing / Copy Pack`.
- Organize sections for App Store, Google Play, Shared Positioning, and Claim Notes.
- Keep text editable and selectable.
- Use simple cards, headings, and labels; avoid screenshot mockups unless separately requested.
- Keep approved copy and draft variants visually distinct.

## Figma Rules

- Preserve approved frames unless replacement is explicitly requested.
- Place new copy boards near existing marketing/store listing work when possible.
- Use existing brand/design language lightly, but prioritize copy readability.
- Use Auto Layout with `Hug contents` for text cards and `Fill container` for full-width sections.
- Validate clipped text, collapsed frames, missing fonts, and unclear section labels.

## Completion Notes

Report:

- Created or updated Figma frame names.
- Which copy version is recommended.
- Which variants are alternatives.
- Any claims that need owner confirmation before store submission.
