# Store Metadata Checklist

Use this checklist for complete App Store and Google Play copy packs.

## App Store

- App name candidates.
- Subtitle candidates.
- Promotional text.
- Keywords and keyword themes.
- Full description.
- What’s new / release notes.
- Claim-risk notes.

## Google Play

- App title candidates.
- Short description candidates.
- Full description.
- Update notes.
- Feature bullets.
- Tags/category notes when useful.
- Listing experiment variants when requested.

## Shared Positioning

- Target user.
- Core promise.
- Key features.
- Differentiators.
- Proof points or product constraints.
- Claims that need confirmation.

## Review Checks

- Copy matches actual or explicitly planned product behavior.
- Platform availability is accurate.
- AI, privacy, subscription, and automation claims are not overstated.
- Short copy is readable at store-listing scan speed.
- Final recommendation is separated from alternatives.
