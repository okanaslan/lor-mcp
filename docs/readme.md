# Agentic Router MCP Server Docs

This folder is the planning and specification home for the Agentic Router MCP
Server.

## Contents

- `roadmap.md`: high-level feature roadmap.
- `feature-specs/`: feature specifications, added as major features are
  defined.

## Working Notes

- Keep feature planning documents under this `docs/` folder.
- Add one focused feature spec per major feature.
- Update the roadmap when a feature spec is added, changed, or completed.
